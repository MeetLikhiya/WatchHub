# WatchHub

WatchHub is a Django-based e-commerce web application for browsing and purchasing watches.

## Features
- User authentication (login and register)
- Watch category listing
- Watch detail page
- Shopping cart
- Checkout system
- Admin watch management

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/MeetLikhiya/WatchHub.git
```

Go to project folder:

```bash
cd WatchHub
```

Run migrations:

```bash
python3 manage.py migrate
```

Run server:

```bash
python3 manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000/
```

## Project Structure

```
config/        → Django project settings
watches/       → Main application
templates/     → HTML templates
static/        → CSS, JS, images
media/         → Uploaded watch images
```
