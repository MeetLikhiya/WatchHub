# WatchHub

WatchHub is a Django-based e-commerce web application for browsing, comparing, and purchasing watches online.

## 📌 Project Description

The E-Commerce Watch Collection Platform is designed to solve the limitations of traditional offline watch retail systems. It provides a centralized digital platform where users can browse, compare, and securely purchase watches. The system also helps in managing inventory and orders efficiently.

## 🚀 Features

- User authentication (Login & Register)
- Watch category listing
- Watch detail page
- Shopping cart
- Wishlist functionality
- Checkout & order management
- Admin panel for product management

## 🛠 Technologies Used

- Python
- Django
- HTML, CSS, JavaScript
- SQLite
